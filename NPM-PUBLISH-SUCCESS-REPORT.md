## 📱 **用户使用方式**

### **方式1: 下载包使用（已验证）**
```bash
# 下载包
npm pack deepcode-cli

# 解压使用
tar -xzf deepcode-cli-1.0.0.tgz
cd package

# 设置API密钥
export DOUBAO_API_KEY="your-api-key"

# 直接使用
node deepcode --ask "你好，豆包！"
```

### **方式2: 全局安装（需管理员权限）**
```bash
# 全局安装
sudo npm install -g deepcode-cli

# 直接使用
deepcode --ask "你好，豆包！"
```

### **方式3: npx使用（推荐）**
```bash
# 无需安装，直接使用
export DOUBAO_API_KEY="your-api-key"
npx deepcode-cli --ask "你好，豆包！"
```
